# Easy Pricing Table with Admin Customization

## Beschreibung

Das Plugin "Easy Pricing Table with Admin Customization" ermöglicht es, moderne Preistabellen in WordPress zu erstellen und zu verwalten. Über den Admin-Bereich können bis zu drei Preistabellen erstellt werden, die auf einer Seite oder in einem Beitrag über einen Shortcode angezeigt werden. Jede Preistabelle kann ein Angebot, einen Preis, eine Beschreibung (in Listenform) und einen Link zu einer Produktseite enthalten. Die Preistabellen sind mit einem ansprechenden, modernen Design versehen, inklusive Schattenboxen und einem responsive Layout.

## Installation

1. **Plugin herunterladen**: Lade die Plugin-Datei von [https://torbenb.info/download](https://torbenb.info/download) herunter.
2. **Plugin hochladen**: Lade den Plugin-Ordner in das Verzeichnis `/wp-content/plugins/` deiner WordPress-Installation hoch.
3. **Aktivieren**: Gehe im WordPress-Dashboard zu **Plugins**, suche nach **Easy Pricing Table with Admin Customization** und klicke auf **Aktivieren**.

## Anwendung

### Shortcode verwenden

Um die Preistabellen auf einer Seite oder einem Beitrag anzuzeigen, verwende den folgenden Shortcode:

```
[easy_pricing_table]
```

Füge diesen Shortcode in den Inhalt einer Seite oder eines Beitrags ein, um die Preistabellen an der gewünschten Stelle darzustellen.

### Preistabellen im Admin-Bereich verwalten

1. Gehe im WordPress-Dashboard zu **Preistabellen Einstellungen**.
2. Dort kannst du bis zu drei Preistabellen erstellen und bearbeiten.
3. Für jede Preistabelle kannst du folgende Informationen angeben:
   - **Angebot**: Der Name des Angebots (z. B. "Basic", "Pro", "Premium").
   - **Preis**: Der Preis des Angebots (z. B. "€9.99/Monat").
   - **Beschreibung**: Eine Liste von Merkmalen oder Leistungen (jede Beschreibung in einer neuen Zeile).
   - **Produktseite URL**: Ein Link zu einer Produktseite, der Benutzer zur Kaufseite weiterleitet.
4. Klicke auf **Speichern**, um die Änderungen zu übernehmen.
